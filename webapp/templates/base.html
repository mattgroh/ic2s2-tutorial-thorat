Instructions

These instructions are presented as a modal popup from the base.html file in the templates folder.

The base.html is an HTML file that calls Javascript and CSS, which give this website its style and feel. We're going to abstract the Javascript and CSS today, so you don't need to do anything with these files. The base.html file also contains the navbar with the About, Instructions, and Consent sections.

If you navigate to the informed-consent.html file, you can see how the base.html file is injected via the `extends "base.html"` into the `block content` that you find on line 140 of base.html.

Now you're ready to start building without a direct example. Can you build an about section?
